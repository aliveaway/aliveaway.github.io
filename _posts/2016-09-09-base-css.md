书写样式  
<h4>1.外部样式</h4>
``` css
<link rel="stylesheeet" type="text/css" src="main.css"/>
``` 
<h4>2.头部样式</h4>
``` css
<head>
<meta http-equiv="Content-Type" content="text/html;charset=UTF-8">
<title>Document</title>
<style type="text/css">
	.main{
		background-color: #999;
		margin: 0
	}
</style>
</head>
```
